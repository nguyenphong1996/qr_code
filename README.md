Ứng dụng quản lý phòng và quét thiết bị mạng, tự động tạo mã QR.

## 📖 Cách sử dụng

### 1. Chọn chi nhánh
Nhấp dropdown chọn chi nhánh ICOOL cần quét.

### 2. Quét mạng
- Nhấp **"ScanQR"** để quét thiết bị trên mạng
- Ứng dụng sẽ tìm tất cả phòng của chi nhánh và tạo mã QR

### 3. Quản lý phòng
Nhấp **"Quản lý thiết bị"** để:
- **Thêm phòng** - Nhập tên phòng (101, 102...) → Nhấp "Thêm"
- **Sửa phòng** - Nhấp vào phòng → Chỉnh sửa
- **Xóa phòng** - Nhấp vào phòng → Xóa
